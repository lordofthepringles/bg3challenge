# BG3 Challenge Generator

A random challenge run generator for Baldur's Gate 3!

 Features
- 340+ unique challenge modifiers
- Difficulty presets (Light, Medium, Extreme)
- Party size presets
- Conflict detection
- Save/Load challenges
- Streamer mode with chat voting
- Dark mode

 How to Use
1. Choose a difficulty or customize your own
2. Click "Generate Challenge"
3. Save or share your challenge code
4. Have fun with your chaotic BG3 run!
